# Prime Names (Medium)
Write a program that takes a string as input and returns whether the value of that string is a prime number. By:
  1. Splitting the string by character
  2. Adding the ASCII values of the characters in the string
  3. Determining if that sum is a prime number

### Input
`BobbyBearcat`
### Output
```
Sum: 1184
NOT PRIME
```
<br>

### Input
`RobertaBearcat`
### Output
```
Sum: 1409
PRIME
```
